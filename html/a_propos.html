Projet Web - TI404

Description du projet

Dans le cadre du module de Programmation Web de ce semestre TI402, nous a été donné le projet suivant :
Concevoir un site internet avec HTLM, CSS, JavaScript. Incluant la stylisation du contenu, l'intéractivité et contanant des tableaux, formulaires.
Avec quelques consignes supplémentaires :
les pages devant être valide W3C, les sources des images devant être citées...

Collaborateurs

Ce projet web a été conçu et développé par William Kuang et Guillaume Pousse. La direction du projet est né de l’envie de proposer un espace où les cinéphiles peuvent découvrir, partager autour des films.
William s’est concentré sur la structure technique et les fonctionnalités interactives (JavaScript), tandis que Guillaume a pris en charge le design, l’ergonomie et l’expérience utilisateur (CSS).
Le résultat : un site fluide, responsive et pensé pour tous les amoureux du 7ᵉ art.
Le code source ainsi que les contributions de chacun est disponible sur le dépôt Github.
